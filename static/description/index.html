Türkiye Sanal POS Ödeme Sistemi

Bu modül, Türkiye'deki tüm bankaların sanal POS sistemlerini Odoo'da kullanmanızı sağlar. mewebstudio/pos projesinden esinlenerek geliştirilmiştir.

Özellikler

Ödeme Tipleri

  • 3D Secure - En güvenli ödeme yöntemi
  • 3D Pay - 3D Secure ile ödeme
  • 3D Host - Banka sayfasında ödeme
  • Non-Secure - 3D olmadan ödeme

Taksit Seçenekleri

  • Kategori bazlı taksit tanımları
  • Banka özel taksit oranları
  • Kampanya taksitleri
  • Komisyon hesaplama
  • Vade farkı hesaplama

İşlem Yönetimi

  • İptal işlemleri
  • İade işlemleri (tam ve kısmi)
  • Durum sorgulama
  • Sipariş tarihçesi
  • Otomatik mutabakat

Kurulum

  1. Modülü Odoo'ya yükleyin
  2. Ayarlar > Türkiye POS menüsünden banka gateway'lerini senkronize edin
  3. Ödeme Sağlayıcıları menüsünden bankanızı ekleyin
  4. API bilgilerinizi girin (Test/Canlı)
  5. Taksit seçeneklerini yapılandırın
  6. Ödeme yöntemini aktif hale getirin
